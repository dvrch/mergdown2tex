# Logo MergDown2TeX

Sauvegardez le logo `logo.png` dans ce dossier.

## Instructions

1. Clic droit sur l'image dans le chat
2. "Enregistrer l'image sous"
3. Sauvegarder comme `docs/assets/logo.png`
4. Puis exécuter :

```bash
git add docs/assets/logo.png
git commit -m "Add logo"
git push
```

## Fichiers requis

| Fichier | Description |
|---|---|
| `logo.png` | Logo principal (fourni par l'utilisateur) |
